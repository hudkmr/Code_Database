ETC Register Dump Testcase

Description:-

This testcase get the register values for the specified address using ETC UART Port.


Test setup:-

  1. Ensure the proper power connection and only one FTDI uart port is connected to your PC
  2. Disconnect if any ZOC terminal is active.
  3. Enter proper output file ending with .csv
Example:-
  1. Start Address  - 16 bit hexadecimal value - eg 0xFA01
  2. End Address   - 16 bit hexadecimal value - eg 0xFAFF
  3. Output File name - "Any_std_name".csv
