This Script file downloads the the gphy firmware into the flash device using ETC UART Port 2
Three Parameters are given passed to script while execution

Example :- 
D:\GIT\Code_Database\Flash_dwnld>python flashdwnld_fast.py
Flash Download Programe is initiated
Ensure the COMPORT is free
Enter the Com port no:30
Enter the Baud Rate:115200
Enter the hex file to be copied to flash memory:etc_fd_8516.hex

Note - the firmware file download is in Big ENDIAN format
